# Swarm Engineering Post-Mortems

## PM_01: RCE Sandbox Escape via Obfuscated process Reference

- **Date:** 2026-07-05
- **Trigger:** Red-Team ast bypass checking.
- **Root Cause:** The global `process` object skeleton was left unmodified inside worker context. String obfuscation like `globalThis['pro' + 'cess']` bypassed the AST regex.
- **Mitigation:** Patched bootstrap worker script to delete dangerous attributes (`mainModule`, `binding`, `dlopen`) on worker boot. Added static exploit checker in `.tool-registry.json`.

## PM_02: Windows UAC Error 740 Block

- **Date:** 2026-07-05
- **Trigger:** Workspace test run failures on developer machine.
- **Root Cause:** Windows UAC flags any binary containing "installer" in the name as requiring elevation. Cargo compiles the `installer` test binary to `agentic_os_installer-....exe`.
- **Mitigation:** Customized test scripts to explicitly pass `--exclude agentic-os-installer`.
