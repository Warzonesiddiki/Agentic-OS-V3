# Environment Variables Reference

This document provides a reference for all environment variables used in the Agentic OS V3.

## Core Environment Variables

- `AGENTIC_OS_API_KEY`: API key for accessing Agentic OS services.
- `AGENTIC_OS_API_URL`: Base URL for the Agentic OS API.
- `AGENTIC_OS_LOG_LEVEL`: Logging level (DEBUG, INFO, WARN, ERROR).

## Database Environment Variables

- `DATABASE_URL`: Connection string for the database.
- `DATABASE_POOL_SIZE`: Number of connections in the database pool.

## Cache Environment Variables

- `REDIS_URL`: Connection string for Redis.
- `CACHE_TTL`: Time-to-live for cache entries in seconds.

## Feature Flags

- `ENABLE_FEATURE_X`: Enable or disable feature X.
- `ENABLE_FEATURE_Y`: Enable or disable feature Y.
