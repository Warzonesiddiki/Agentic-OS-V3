## Legend

- **ALREADY_BUILT** — the feature and its core functions largely exist today (file:line evidence cited). A later agent can verify and harden rather than build.
- **PARTIAL** — pieces exist but the task's acceptance/integration gap remains (e.g. function built but not wired, or route/table missing).
- **NEW** — nothing exists; must be built from scratch (target file does not exist).
