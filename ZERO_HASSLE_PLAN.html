  2. ✅ Single start script — start.bat / start.sh that checks for Node.js, installs deps from local cache, prompts for API key, starts server
  3. ✅ Auto-migrate — Server auto-runs Drizzle push on first start
  4. ✅ Default config — Ship with sensible defaults, only require API key
  5. ✅ Rust binary for advanced users — The crates we're building produce a native CLI that works without Node.js

The Rust binary path (Phase C) is the long-term goal — a true single-file AI operating system.
